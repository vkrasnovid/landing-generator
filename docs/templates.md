# Templates & Customization

Details on the 3 built-in templates, color presets, and hero photo library.

## Templates Overview

### Classic

The default template. Full-screen hero image with dark overlay, followed by structured content sections.

**Sections:**
- Hero — full-viewport background image, company name, slogan, CTA button
- About — two-column layout (text left, image right)
- Services — 2-column card grid with icons
- Team — member cards with name, role, and bio
- Contacts — icon-based contact cards
- Footer — company name and copyright

**Best for:** general business sites, agencies, service companies.

### Minimal

Clean, modern design focused on typography and whitespace.

**Sections:**
- Hero — text-only, large headline (64px+), no background image
- About — single-column centered text
- Services — simple list with icons
- Team — minimal cards
- Contacts — clean layout
- Footer — minimal copyright line

**Best for:** creative studios, portfolios, personal brands.

### Corporate

Formal, data-driven layout with a professional tone.

**Sections:**
- Hero — structured header with navigation bar
- Key Facts — number bar ("10+ years", "500+ clients", "24/7 support")
- About — two-column with company photo
- Services — table format (icon, name, description columns)
- Team — professional grid cards
- Contacts — structured contact cards
- Footer — contrast bar with copyright

**Best for:** law firms, consulting companies, finance, B2B.

## Color Presets

Five built-in color schemes:

| Preset | Primary Color | Hex |
|--------|--------------|-----|
| Purple (default) | Violet | `#6c63ff` |
| Blue | Classic blue | `#2196F3` |
| Green | Emerald | `#4CAF50` |
| Red | Crimson | `#f44336` |
| Dark | Charcoal | `#333333` |

To use a **custom color**, use the color picker or type a hex value directly in Step 5.

The primary color affects:
- Hero overlay and CTA buttons
- Section accents and headings
- Link colors and hover states
- Card borders and highlights

## Hero Photo Library

Built-in Unsplash photos organized by category:

| Category | Examples |
|----------|----------|
| Business | Office spaces, meetings, teamwork |
| Law | Courtrooms, legal books, scales of justice |
| Medicine | Hospitals, medical equipment, healthcare |
| IT | Code screens, servers, tech workspaces |
| Food | Restaurants, kitchens, dining |
| Auto | Cars, dealerships, automotive |

Each category has 3 curated photos. Click a photo to set it as the hero background (used in Classic and Corporate templates). Minimal template does not use hero photos.

## Responsive Design

All templates include mobile breakpoints in their inline CSS:

- **Desktop:** full two-column/grid layouts
- **Mobile (< 768px):** single-column stacked layout, adjusted font sizes, simplified navigation

No media query customization is needed — responsive behavior is built into each template.
