# Stories: Landing Page Generator

## Story 1: Форма ввода данных
**Описание:** Многошаговая форма (wizard): название компании, описание, услуги (до 8), команда, контакты, цветовая схема.
**Acceptance criteria:**
- [ ] Шаг 1: Название компании, слоган, описание (текст)
- [ ] Шаг 2: Услуги — до 8 карточек (название + описание + иконка)
- [ ] Шаг 3: Команда — до 6 человек (имя, должность, описание)
- [ ] Шаг 4: Контакты (телефон, email, адрес, соцсети)
- [ ] Шаг 5: Стиль — выбор цветовой схемы (3-5 пресетов) или custom цвета
- [ ] Прогресс-бар шагов
- [ ] Валидация полей
- [ ] Данные сохраняются в state между шагами
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 3 часа

## Story 2: Шаблоны лендингов
**Описание:** 3 готовых шаблона: "Классический" (как у коллегии адвокатов), "Минималистичный", "Корпоративный". Каждый — полноценный HTML/CSS с секциями hero, about, services, team, contacts, footer.
**Acceptance criteria:**
- [ ] Шаблон "Классический": hero с overlay, about с фото, services grid, team cards, reviews, contacts
- [ ] Шаблон "Минималистичный": чистый белый, большие шрифты, минимум декора
- [ ] Шаблон "Корпоративный": строгий, синие тона, таблицы, факты в цифрах
- [ ] Каждый шаблон responsive (mobile + desktop)
- [ ] Шаблоны как JS template literals с плейсхолдерами
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 4 часа

## Story 3: Генератор HTML
**Описание:** Движок генерации: берёт данные из формы + выбранный шаблон → генерирует готовый HTML-файл с inline CSS/JS.
**Acceptance criteria:**
- [ ] Подстановка данных в шаблон (название, услуги, контакты и т.д.)
- [ ] Применение выбранной цветовой схемы
- [ ] Подстановка выбранных иконок для услуг
- [ ] Генерация мета-тегов (title, description, OG tags)
- [ ] Выходной файл — самодостаточный index.html (все стили inline)
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 2 часа

## Story 4: Предпросмотр и скачивание
**Описание:** Живой предпросмотр лендинга в iframe. Кнопки "Скачать HTML" и "Скопировать код".
**Acceptance criteria:**
- [ ] Iframe с предпросмотром (обновляется при смене данных/шаблона)
- [ ] Кнопка "Скачать" — скачивает index.html файл
- [ ] Кнопка "Копировать код" — копирует HTML в буфер
- [ ] Переключатель desktop/mobile preview (ширина iframe)
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 2 часа

## Story 5: Библиотека иконок и изображений
**Описание:** Встроенная библиотека иконок (Font Awesome / emoji) для услуг. Unsplash фото для hero/about по категориям (бизнес, юриспруденция, медицина, IT, еда, авто).
**Acceptance criteria:**
- [ ] Выбор иконки для каждой услуги (минимум 30 иконок)
- [ ] Выбор hero-фото из категорий (6+ категорий, 3+ фото каждая)
- [ ] Превью иконок и фото при выборе
**Агент:** developer-frontend
**Приоритет:** P2
**Оценка:** 2 часа

## Story 6: Дизайн генератора
**Описание:** Красивый UI самого генератора: тёмная тема, плавные переходы между шагами, анимации.
**Acceptance criteria:**
- [ ] Тёмная тема генератора
- [ ] Плавные CSS transitions между шагами
- [ ] Анимация генерации (прогресс-бар или spinner)
- [ ] Все тексты на русском
- [ ] Mobile-friendly
**Агент:** developer-frontend
**Приоритет:** P2
**Оценка:** 1.5 часа

## Story 7: E2E тестирование
**Описание:** Полный цикл: заполнить форму → выбрать шаблон → предпросмотр → скачать → открыть HTML → проверить.
**Acceptance criteria:**
- [ ] Форма заполняется корректно (все шаги)
- [ ] Все 3 шаблона генерируют валидный HTML
- [ ] Скачанный файл открывается в браузере
- [ ] Все данные из формы присутствуют в сгенерированном HTML
- [ ] Responsive сгенерированных лендингов
- [ ] Нет console errors
**Агент:** tester
**Приоритет:** P1
**Оценка:** 1.5 часа

---

**Итого:** 7 stories, ~16 часов
